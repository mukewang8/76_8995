<h3>课程目标：</h3>
<p>差旅管理系统主要是针对于大型企业的员工出差进行统一管理调配的应用系统，同时利用此系统可以很好的解决出差冲突的问题，也可以针对于企业的资源进行更加合理的规划。该项目核心模块：员工管理模块、出差申请模块、差旅审核模块。</p>
